# Конфигурационный трансформер (Вариант 22)

Инструмент командной строки для преобразования учебного конфигурационного языка в JSON.

## Возможности языка (Вариант 22)

- **Числа**: `[+-]?\d+` (целые числа со знаком)
- **Строки**: `'текст'` (в одинарных кавычках)
- **Массивы**: `[значение1 значение2 ...]` (без запятых)
- **Словари**: `@{ ключ = значение; ... }`
- **Константы**: `set имя = значение`
- **Выражения**: `$имя + 1$` (поддерживается сложение и `len()`)
- **Комментарии**: `/# многострочный комментарий #/`
